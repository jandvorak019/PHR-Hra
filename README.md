# PHR-Hra

1. Základní koncepce
3D FPA (First Person Adventure) Godot hra Sunny Field je semi-horrorová First Person Adventure hra kde člověk prozkoumává mapu a samotnou hru.
Hráč prozkoumává mapu a objevuje zajímavé a zvláštní věci a různé postavy. Finální hra se bude hlavně zabívat movementem hráče a skákáním na platformách a také možná combatu s enepřítelem. Hra bude rozdělená na levely.

3. Level design
Level 1 (prototyp) se skládá z open world mapy s travnatými kopci, horami a neviditelnými zdmi kolem celé mapy. Hráč začíná ze spawnpoint budovy. 
Hráč se dokáže pohybovat, sprintovat i skákat. Na hráče funguje i skluz (slide).

### Ovládací prvky projektu:
- Pohybujte pomocí `W` `A` `S` `D` nebo `↑` `←` `↓` `→`
- Skok s `Space`. Sprintujte pomocí `Shift`
- Odemkněte/Zamkněte myš pomocí `Shift + F1`
- Zavřete hru pomocí `Esc`
